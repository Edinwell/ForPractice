    「Windows Vista」で動作させる場合
    Windows系はTomcatの対応OSであればこのマニュアル通りで動作できるかと思いますが、
    Vistaで動かすと、「unable to open the service 'Tomcat7'」と言うダイアログが出てくると思います。
    この場合、「コントロールパネル」→「ユーザーアカウント」→「ユーザーアカウント制御の有効化または無効化」→
    ユーザーアカウント制御(UAC)を使ってコンピュータの保護に役立てる のチェックボックスをはずして、
    OKを押して再起動すると動かせるようになります。

    また、上図の「Manager　App」を押して下さい。

    終了方法は「Configure Tomcat」を押下して、「stop」を押下すると終了します。
