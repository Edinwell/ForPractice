    JDKとJREの違いについて

    インストール後、C:\Program Files\Javaの中には「jdk・・」と、「jre・・」と２つのフォルダが作成されますが、
    JDK は Java SE Development Kit の略で、JRE は Java Runtime Environment の略です。
    JDK はコンパイラ、JRE は Java のプログラムを実行するための JVM です。
    JVMとはJava Virtual Machineの略で、Java プログラムを実行するための仮想マシンの事。
    簡単に説明すると、JVMを入れれば、どんなPC環境下でもJava プログラムを動かすことができます。


    ２. 環境変数の設定

      「コントロールパネル」→「システムとセキュリティ」→「システム」→「システムの詳細設定」→「環境変数」と選択して、
      システム環境変数に下記のものを追加していきます。

      PATH ：　C:\Program Files\Java\jdk1.8.0_45\bin

      変数値の最後にセミコロンを打って追加します。
      "・・・;C:\Program Files\Java\jdk1.8.0_45\bin"
      元から入ってる変数値は消さないように注意！



      既に起動しているコマンドプロンプト等のプログラムは、プログラムを再起動しないと
      設定が反映されません。
      (パソコンを再起動する必要はありません)
