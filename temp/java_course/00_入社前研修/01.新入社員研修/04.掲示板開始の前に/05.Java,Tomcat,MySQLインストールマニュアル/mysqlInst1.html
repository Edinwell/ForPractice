    ３、my.iniの編集
      Installerを使ってインストールした場合には、C:\ProgramData\MySQL\MySQL Server 5.6下に
      自動でmy.ini（構成設定ファイル）が生成されます。

      ProgramDataフォルダは Windows の標準の設定では隠れていて表示されないようになっています。
      コントロールパネル→デスクトップのカスタマイズ→フォルダーオプションの
      「隠しファイル、隠しフォルダー、および隠しドライブを表示する。」
      をチェックし適用ボタンを押すことで、表示させることができます。



      MySQL上で使えるエンコードを変更するため、一部編集します。
      編集箇所は、「default-character-set=utf8」と書かれている箇所を編集します。



      上の図で「utf8」を「cp932」に書き直して下さい。全部で２箇所（図の箇所）あります。

    ４、環境変数の設定
      「コントロールパネル」→「システムとセキュリティ」→「システム」→「システムの詳細設定」→「環境変数」と選択して、
      システム環境変数に下記のものを追加していきます。

      PATH：C:\Program Files\MySQL\MySQL Server 5.6\bin

      変数値の最後にセミコロンを打って追加する。
      "・・・;C:\Program Files\MySQL\MySQL Server 5.6\bin")
      元から入ってる変数値は消さないように注意！




    ５、MySQLコマンドクライアントの使用方法
      パソコン起動と同時にMySQL Serverも起動する設定をしました。（上のインストール時）
      ここでは既に起動されているMySQLのコマンドクライアントの使用方法を説明します。

      コマンドラインで「mysql -u root -p」と入力し, エンターを押すと、"Enter password"と表示されます。
      ここに上で登録したパスワード（ここでは"root"）を入力しエンターを押します。
