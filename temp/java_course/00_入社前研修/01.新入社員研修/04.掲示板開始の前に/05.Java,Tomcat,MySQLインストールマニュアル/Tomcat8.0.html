3. Tomcat の起動方法と終了方法
「Windows Vista」で動作させる場合

Windows系はTomcatの対応OSであればこのマニュアル通りで動作できるかと思いますが、
Vistaで動かすと、「unable to open the service 'Tomcat8'」と言うダイアログが出てくると思います。
この場合、「コントロールパネル」→「ユーザーアカウント」→「ユーザーアカウント制御の有効化または無効化」→
ユーザーアカウント制御(UAC)を使ってコンピュータの保護に役立てる のチェックボックスをはずして、
OKを押して再起動すると動かせるようになります。

ここから、Tomcatを起動させていきます。
起動方法は「スタート」→「すべてのプログラム」→「Apache Tomcat 8.0」と進み、「Configure Tomcat」を押下して下さい。



↓

下図が現れるので「start」を押下して下さい。
「stop」を押下すると終了します。




↓

起動に成功したかの確認は、「http://localhost:8080/」とブラウザに入力して下さい。


下記のページが現れれば起動成功です。




次に、上図の「Manager　App」を押して下さい。

ユーザー名と、パスワードを聞かれるので、前に設定したユーザー名=insight, パスワード=insight
を入力してログインできることも確認しておいて下さい。下記のページが現れます。
ユーザー名と、パスワードを忘れてしまった場合は、Tomcat をインストールしたディレクトリ内の「conf」フォルダの「tomcat-users.xml」を開くと書いてあります。





これ以降本ドキュメントでは、Tomcat をインストールしたディレクトリ C:\tomcat 8.0 を %CATALINA_HOME% と記載します。
